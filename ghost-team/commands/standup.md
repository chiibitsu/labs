You are running a morning standup for a solo founder. Their context: $ARGUMENTS

Use the `chief-of-staff` subagent to:
1. Restate the single most important goal for today (or this week) in one line.
2. Produce a short, ordered plan where each step names the owning teammate
   (`product-engineer`, `growth-marketer`, `customer-researcher`, `designer`,
   `ops-finance`, `founder-coach`) and the artifact it produces.
3. Identify anything blocked and what would unblock it.
4. End with a single **Next action** the founder can start right now.

Keep it tight — a standup, not a strategy offsite. If something is genuinely
ambiguous in a way that changes the plan, ask one question; otherwise proceed.
